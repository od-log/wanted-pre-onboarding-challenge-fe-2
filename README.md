## 프로젝트의 실행 방법

1. 아래 명령어로 저장소를 복제해주세요.
```
git clone https://github.com/od-log/wanted-pre-onboarding-frontend.git
```
2. 아래 명령어로 npm을 설치해주세요.
```
npm install
```
3. 아래 명령어로 npm 또는 yarn을 실행해주세요.
```
npm start
```
또는
```
yarn start
```
