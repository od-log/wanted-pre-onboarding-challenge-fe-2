## 프로젝트의 실행 방법
1. root 폴더에 .env 파일 생성 후 아래 코드를 입력해 주세요.
```
REACT_APP_BASE_URL = https://pre-onboarding-selection-task.shop 
```
2. 아래 명령어로 npm을 설치해주세요.
```
npm install
```
3. 아래 명령어로 npm을 실행해주세요.
```
npm start
```
